# Certified Agentic AI & Robotics Engineer (CAARE) Syllabus
## Context & Standards for Project Al-Buraq

### Program Philosophy
"From Code Writer to AI Orchestrator."
We must transition from simple coding to architecting reasoning software agents.

### Core Methodologies (MUST FOLLOW)
1. **Spec-Driven Development (SDD):**
   - Use **Spec-Kit Plus**.
   - Specifications must be executable and enforce rules (like Halal/Commission).
   - Code must be generated FROM specs.

2. **Agentic AI Architecture (Quarter 3):**
   - **Brain:** OpenAI Agents SDK (Stateful agents, Handoff patterns).
   - **Tools:** Model Context Protocol (MCP) for connecting to data/APIs.
   - **Data Layer:** Vector Databases (RAG) + Relational DBs.

3. **User Interface (Quarter 3 - Week 12):**
   - **Framework:** OpenAI ChatKit.
   - **Frontend:** Next.js/React.
   - **Goal:** Conversational UI where humans and agents collaborate.

### Technical Stack Requirements
- **Language:** Python 3.12 (Backend), TypeScript/Next.js (Frontend).
- **AI Models:** GPT-4o, Claude 3.5 Sonnet.
- **Environment:** Antigravity IDE / Cursor / Claude Code CLI.
- **Deployment:** Cloud-Native (eventually Kubernetes/Dapr in Q4).

### Project: Al-Buraq Dispatcher
This project must align with the above standards.
- Current Status: Python Monolith with basic functions.
- Target Status: Multi-Agent System using OpenAI Agents SDK + ChatKit UI.
